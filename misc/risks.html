Risks

Why is this protocol so important?

This protocol, with its resilience allow organization and companies to securely transport sensitive data across the internet, including Vital data such as patient health records from medical facilities. However, some negatives about the use of this procol need to be consider: Connectivity loss, Component failure or packet loss might cause the system to be vulnerable and expose the data to unauthorized access.
Despite these negative, if the potential risks can be mitigated effectively, the protocol can be immensely successful and greatly benefit society.

Why do we need Data Security?

The need for data security in today’s world cannot be overstated. A lack of security can lead to high-profile hacks, loss of data, financial losses and fines which can damage an organization’s reputation. Implementing data security techniques, could enhances an organization’s reputation and instills trust among current and future customers.
